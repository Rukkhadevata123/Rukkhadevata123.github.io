Welcome to Yoimiya's Minecraft Server Setup Guide!

1. Introduction

In this guide, we'll walk through the steps of setting up your own Java Edition server using the default server software that Mojang Studios distributes for free. For more details, you might want to visit the official Minecraft Wiki.

2. Setting Up

Use the Minecraft server port number as the starting, ending, internal, and external port number. By default, the port number is "25565".

3. Notes

Minecraft servers allow players to play online or via a local area network with other people. Also, remember to use this script as a guideline for setting up a startup script for the Minecraft server. It should be in the player's executable $PATH.
